Sobre o Projeto

Sistema completo de gestão escolar desenvolvido para facilitar a administração de instituições de ensino. O projeto inclui funcionalidades para gestão de alunos, professores, turmas, notas e frequências.

Funcionalidades

  • Cadastro de alunos e professores
  • Gestão de turmas e horários
  • Lançamento de notas e faltas
  • Emissão de boletins e relatórios
  • Comunicação com pais e responsáveis
  • Calendário escolar integrado

Tecnologias

Java
